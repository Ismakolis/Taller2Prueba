-----Proyecto Angular - Formulario con envío por POST

Este proyecto consiste en un formulario construido con Angular que envía los datos a través del método HTTP POST.

----------Requisitos

- Node.js y npm instalados
- Angular CLI (`npm install -g @angular/cli`)
- Git (opcional, si clonas el repositorio)

----------Instalación del proyecto

1. Clona el repositorio:
   git clone https://github.com/TU_USUARIO/TU_REPOSITORIO.git

2. Ingresa a la carpeta del proyecto:
   cd nombre-del-proyecto

3. Instala las dependencias:
   npm install

4. Ejecuta el proyecto:
   ng serve

5. Abre tu navegador y visita:
   http://localhost:4200

----------Descripción

- El formulario se encuentra en el componente `Login`.
- Al completar y enviar el formulario, los datos se envían mediante una solicitud POST.
- Por ahora, el proyecto simula el envío de datos no cuenta con un backen configurado.
